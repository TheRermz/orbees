# ORBEES

Desenvolvimento de uma plataforma web de monitoramento financeiro pessoal que automatiza a importação e categorização de transações bancárias via extrato OFX/CSV, com módulos complementares de educação financeira, controle em grupo e assistente de alertas automáticos.

## Stacks Utilizadas

Para o front-end foi decidido utilizar React com Typescript e para o back-end C# com ASP.NET
